# questoes01-12


**Exercícios – Revisão de Python **
1. Cadastro de Alunos (Completo)
Crie um programa que:

Peça ao usuário o nome, idade e uma lista de notas (separadas por vírgula).
Armazene tudo em um dicionário.
Use uma função para calcular a média.
Salve o resultado em um arquivo chamado aluno.txt.
Use uma condicional para dizer se o aluno está aprovado (média ≥ 7).
Use tupla para armazenar os nomes das matérias (ex.: ("Matemática", "Português")).
Use set para remover notas repetidas antes de calcular a média.
2. Verificando Presença em um Evento
Crie uma lista com 5 nomes. Depois:

Peça uma entrada de nome ao usuário.
Converta para set a lista para remover duplicatas.
Use um if para verificar se o nome está presente.
Use um for para mostrar todos os nomes no evento.
Crie uma função chamada adicionar_nome que adiciona o nome na lista somente se ele não existir.
Salve a lista final em um arquivo presenca.txt.
3. Sistema de Estoque Simples
Crie um dicionário onde a chave é o nome do produto e o valor é a quantidade.
Peça ao usuário para digitar um produto e quantidade.
Use uma função que atualize o estoque.
Se o produto não existir, adicione. Caso contrário, incremente.
Use while para permitir várias inserções até o usuário digitar "sair".
Mostre ao final os produtos em ordem alfabética (use sorted do módulo builtins).
Salve o dicionário em um arquivo estoque.txt.
4. Sistema de Login (Simples e Didático)
Crie um programa que:

Armazene logins e senhas em um dicionário.

Peça ao usuário login e senha.

Use booleanos para indicar se o login é válido.

Use uma condição que:

Mostra "Login bem-sucedido"
Ou "Acesso negado"
Registre a tentativa de login em um arquivo log.txt.

Use uma tupla com os tipos de usuário: ("admin", "aluno", "visitante").

Leia quantas tentativas foram feitas usando um arquivo externo.

5. Analisador de Texto
Crie um programa que:

Peça ao usuário um texto.

Transforme esse texto em uma lista de palavras.

Transforme a lista em set para descobrir quantas palavras diferentes existem.

Salve no arquivo analise.txt:

Número total de palavras
Número de palavras únicas
Use uma função contar_palavras(texto) para retornar estes valores.

Use for e if para contar quantas palavras começam com vogal.

Use uma condicional para dizer se o texto é "longo" (mais de 20 palavras).

Use um módulo da biblioteca (string) para ignorar pontuação.
